Start by gathering the supplies. Select the weave, color of paracord and nylon webbing.

Sow the nylon webbing, attach the clip as well as the adjuster.

Start the weave. Start with one end connected to the end of the nylon webbing and the other side attached to the last connector.

Finish up the weave, gather all the excess paracord trim it and burn it. Make sure everything is nice and even that it looks good.
